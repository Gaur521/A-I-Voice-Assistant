# AI Voice Assistant

## 📌 Project Overview

This is an AI-powered Voice Assistant developed using **Python, AI/ML, and Google Bard API**. It understands **natural language commands**, performs tasks, and improves responses using machine learning. This assistant can be used for personal assistance, productivity enhancement, and smart device control.

## 🚀 Features

- **Speech Recognition** 🎙️ (Converts voice to text)
- **AI-Powered Responses** 🤖 (Google Bard API integration)
- **Task Automation** ⚡ (Opens Notepad, fetches time, and more!)
- **Text-to-Speech** 🔊 (Speaks responses back to the user)
- **Machine Learning Adaptation** 📈 (Improves over time)

## 🛠️ Technologies Used

- **Python** 🐍
- **SpeechRecognition** (Voice input processing)
- **Pyttsx3** (Text-to-Speech conversion)
- **Google Bard API** (NLP processing)
- **OS & Datetime Libraries** (System automation)

## 📂 Project Structure

```
AI-Voice-Assistant/
│── main.py           # Main script
│── speech_module.py  # Handles speech recognition & text-to-speech
│── task_executor.py  # Executes commands & fetches AI responses
│── requirements.txt  # Dependencies
│── README.md         # Project documentation
```

## ⚙️ Setup Instructions

1. **Clone the Repository**

   ```bash
   git clone https://github.com/Gaur521/AI-Voice-Assistant.git
   cd AI-Voice-Assistant
   ```

2. **Install Dependencies**

   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Assistant**

   ```bash
   python main.py
   ```
